1. <html> tag principale che contiene tutto il doc html
  2. <head> contiene informazioni sulla pagina che non vengono visualizzati
  3. <body> contiene il contenuto della pagina
  4. <h1> tag di head (font grande)
  5. <p> paragrafo
  6. <b> o <strong> grassetto
  7. <sup> testo sopraelevato
  8. <img> per inserire immagini
  9. <ol> lista ordinata
  10. <ul> lista non ordinata
  11. <li> elemento di una lista
  12. <a> link esterno
